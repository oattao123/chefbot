### Installation

_You must have local python  docker and  git then you can do this._

1. Clone the repo
   ```sh
   git clone https://github.com/oattao123/chefbot.git
   ```
2. Install docker 
   ```sh
   cd chefbot
   docker build -t pychef .
   ```
3. Running docker
   ```sh
   docker run pychef
   ```
